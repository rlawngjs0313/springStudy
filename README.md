# 📋 Commit Message Convention (이모지 + 소문자 적용)
| Tag            | Description             |
|----------------|-------------------------|
| `✨ [feat]`     | 새로운 기능 추가        |
| `🐛 [fix]`     | 버그 수정              |
| `📝 [docs]`    | 문서 추가, 수정, 삭제  |
| `🧪 [test]`    | 테스트 코드 추가, 수정, 삭제 |
| `🎨 [style]`   | 코드 형식 변경         |
| `🔨 [refactor]` | 코드 리팩토링         |
| `⚡ [perf]`     | 성능 개선              |
| `🏗️ [build]`  | 빌드 관련 변경사항     |
| `⚙️ [ci]`      | CI 관련 설정 수정     |
| `🚀 [chore]`   | 기타 변경사항          |
